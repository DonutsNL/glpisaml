# phpsaml2

Rewrite of phpSaml plugin by Derrick Smith for:
- PHP8 support
- GLPI 10+ support
- PSR4 support
- Use concern based class structure
- Focus on code readability
- SONAR LINTING
- Improve community support
- Composer support for updating libraries
